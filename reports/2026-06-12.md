### 1. ARB：基于 C++23 的可微刚体动力学仿真库 [🔗](https://github.com/wbyates777/Articulated-Rigid-Body)
> **来源**：Hacker News / GitHub

- **一句话总结**：一个紧凑的 C++23 刚体动力学仿真库，实现了空间代数、ABA/CRBA/RNEA 算法及端到端自动微分，与工业级 RBDL 基准性能差距在个位数百分比以内。
- **核心要点**：
  - 实现了三种核心动力学算法：Articulated Body Algorithm（ABA，O(N) 前向动力学）、Composite Rigid Body Algorithm（CRBA，O(N²) 求解关节空间惯量矩阵）、Recursive Newton-Euler Algorithm（RNEA，O(N) 逆动力学）
  - 基于空间代数（6D 运动和力向量），将代数复杂度比传统 3D 向量表示降低约 4 倍
  - 通过 autodiff 库实现从代数到 ABA/RNEA 的全流程自动微分，支持系统辨识、模型预测控制等高级应用
  - 碰撞检测支持 GJK/EPA，采用 Projected Gauss-Seidel（PGS）求解器和冲动缓存（warm starting）实现稳定的接触解析
  - 仅依赖 STL + GLM（头文件），替换 Eigen3 为 GLM 以降低体积；支持 C++23（C++20 可移除部分 constexpr）
  - 与 RBDL v3.3.1 性能对比：ABA 快 1.31%，CRBA 慢 4.74%，RNEA 慢 6.09%（SIMD 优化后）
  - 完整支持 URDF 格式导入手，已验证 ur5、iiwa7/14、tiago_dual 等工业机器人模型
  - 支持 Revolute、Prismatic、Spherical、Helical、Planar、Floating Base 等多种关节类型

### 2. Quilt.cpp：C++20 实现的补丁管理工具 [🔗](https://github.com/skeeto/quilt.cpp)
> **来源**：Hacker News / GitHub

- **一句话总结**：知名 C 语言黑客 skeeto 用 C++20 从零重写了 Quilt 补丁管理工具，内嵌 diff/patch 引擎，独立可执行，无需外部依赖。
- **核心要点**：
  - 纯 C++20 实现，单文件 amalgamation 后可 `c++ -std=c++20 -o quilt.exe quilt.cpp -lshell32` 直接编译
  - 内建 diff 引擎，支持 Myers、Minimal、Patience、Histogram 四种算法（与 `git diff --diff-algorithm` 对齐）
  - 内建 patch 引擎，支持模糊匹配（fuzz 1-3）、冲突标记生成、补丁拒绝输出
  - `quilt mail` 输出兼容 `git format-patch --stdout` / `git am` 格式，适合现代工作流
  - 实现了类 shell 的环境变量拆分器（`QUILT_*_ARGS`），支持单引号、双引号、`$VAR`/`${VAR}` 展开
  - 包含 libFuzzer 模糊测试 harness：patch 解析、shell 拆分、diff-patch 往返正确性验证、差异测试（与系统 quilt 对比）
  - 所有代码由 AI（Claude Code）生成，测试先行的开发方法论——README 外每一行都是 AI 编写，测试来自 Quilt 文档和教程
  - 项目持续 4 天完成，含 fuzz 测试和差异测试，质量控制严谨
