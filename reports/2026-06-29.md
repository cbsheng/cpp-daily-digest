### 2. Show HN: Tsar-MCP — How to Create an MCPServer Aspect in C/C++ [🔗](https://ibm.github.io/tsar-mcp/mcp/doc/MCPServer_AspectGuide.html)
> **来源**：IBM GitHub (tsar-mcp)

- **核心要点**：
  - **零外部依赖**：纯 C（C++ 兼容）实现，无 Python/node.js 运行时要求，适合边缘设备

  - 支持同步/异步两种工具模式，内置测试用例框架（NULL 终止的 JSON 请求数组）
  - 编译为独立可执行文件，部署简单
  - 完整的构建系统与 VSCode 注册指南，适合企业团队快速上手
